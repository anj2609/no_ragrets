Responsive Typeracer using Flutter & Node.js and learned about Socket.io, Express, MongoDB, Singleton Pattern Implementation along with Provider to manage the state of the build!

Features:
1. Create/Join Room with validation
2. Copy Room Code
3. Displaying Typed Words, Current Word & Words to be Typed
4. Real Time Progress of Users in Same Room
5. Leaderboard displaying User's WPM
6. Responsive UI
